## See

> [From](https://github.com/mbadolato/iTerm2-Color-Schemes/blob/master/windowsterminal/Gruvbox%20Dark.json)

## Code Fragment

* Theme

> `theme`

```json
{
    "theme": "dark"
}
```

* Color Scheme

> `schemes`

```json
"schemes": [
    {
        "name": "Gruvbox Dark",
        "black": "#1e1e1e",
        "red": "#bb3138",
```

* Profile
  
> `profiles.defaults`

```jsonc
{
    "defaults": {
        // Put settings here that you want to apply to all profiles.
        "background": "#171717",
        "acrylicOpacity": 0.8,
        "useAcrylic": true,
        "colorScheme": "Gruvbox Dark"
    }
}
```
